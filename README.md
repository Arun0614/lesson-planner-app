# Lesson Planner App

An AI-powered lesson planning web app built with React + Vite.

## Features

- Generate lesson plans instantly
- Add class notes and extra context
- Clean responsive UI
- Fast React frontend
- AI-assisted planning workflow

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## Installation

```bash
npm install
npm run dev
```
